## Objetivo del Proyecto

Implementar un sistema de colas de mensajes para procesamiento asíncrono escalable.

## Lo que Aprendí

- Arquitectura de message brokers
- Implementación de workers paralelos
- Estrategias de retry y dead letter queues
- Manejo de backpressure
- Monitoreo y observabilidad

## Tecnologías Utilizadas

- RabbitMQ / Redis Queue
- Worker processes
- Dead Letter Queues
- Monitoring tools
