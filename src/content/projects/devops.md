## Objetivo del Proyecto

Implementar un pipeline completo de CI/CD y despliegue en producción.

## Lo que Aprendí

- Containerización de aplicaciones con Docker
- Orquestación con Kubernetes
- Configuración de CI/CD pipelines
- Estrategias de deployment (blue-green, canary)
- Monitoreo y logging en producción
- Infrastructure as Code

## Tecnologías Utilizadas

- Docker y Docker Compose
- Kubernetes
- GitHub Actions / GitLab CI
- Prometheus y Grafana
- ELK Stack para logging
