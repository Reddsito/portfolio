## Objetivo del Proyecto

Implementar un sistema completo de autenticación seguro y escalable.

## Lo que Aprendí

- Generación y validación de JWT tokens
- Refresh token rotation para mayor seguridad
- Hashing de passwords con bcrypt y salt
- Gestión segura de sesiones
- Protección contra ataques comunes (XSS, CSRF)

## Tecnologías Utilizadas

- JWT (JSON Web Tokens)
- bcrypt para hashing
- Redis para almacenamiento de sesiones
- Rate limiting para prevenir brute force
