## Objetivo del Proyecto

Construir un servidor HTTP desde cero para comprender los fundamentos de los protocolos de red.

## Lo que Aprendí

- Manejo de sockets TCP/IP a bajo nivel
- Protocolo HTTP y sus especificaciones
- Gestión de conexiones concurrentes
- Parseo de requests y generación de responses

## Tecnologías Utilizadas

- Lenguaje: Python/Node.js
- Sockets nativos
- Implementación del protocolo HTTP/1.1
